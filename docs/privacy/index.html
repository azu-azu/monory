プライバシーポリシー

最終更新日: 2026年6月29日

monory（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。

収集する情報

本アプリは、外部サーバーへの個人情報の送信・収集を一切行いません。すべてのデータはお使いのデバイス内にのみ保存されます。

  • 映画ログ（タイトル・評価・感想・鑑賞日など）: すべてデバイス内にのみ保存されます。
  • 写真（チケット画像など）: 記録に添付した写真はデバイス内にのみ保存されます。

ネットワーク通信

本アプリが行うネットワーク通信は以下の場合に限られます。

  • TMDB API への問い合わせ: 映画情報（タイトル・あらすじ・ポスター画像など）を取得するために The Movie Database (TMDB) の API を使用します。送信されるのは映画の検索キーワードまたは映画 ID のみで、個人情報は含まれません。

第三者への情報提供

本アプリは、ユーザーの情報を第三者に販売・提供・開示することはありません。

アナリティクス・広告

本アプリはアナリティクスツール・広告 SDK・クラッシュレポートツールを一切使用していません。

データの管理

アプリをアンインストールすることで、デバイス内のすべてのデータを削除できます。また、設定画面からデータを JSON 形式でエクスポートおよびインポートできます。
